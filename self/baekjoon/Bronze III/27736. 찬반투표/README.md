# [찬반투표 [서브태스크]](https://www.acmicpc.net/problem/27736)

| 시간 제한 | 메모리 제한 | 제출 | 정답 | 맞힌 사람 | 정답 비율 |
| --- | --- | --- | --- | --- | --- |
| 2 초 | 512 MB | 2316 | 940 | 898 | 42.339% |

## 문제

중앙대학교에서 재학생을 대상으로 하는 어떤 찬반투표가 치러졌다. 모든 재학생은 각자 찬성이나 반대, 혹은 기권 중 하나로 투표에 응답하였다.

해당 투표에서 찬성이 반대보다 많으면 투표가 통과된다. 반대가 찬성보다 많거나, 반대와 찬성의 수가 동일하다면 투표는 통과되지 않는다. 단, 기권한 사람이 재학생의 절반 이상이라면 찬성과 반대의 수와 관계없이 항상 투표는 무효 처리된다.

재학생들의 투표 내역을 입력받아 찬반투표의 결과를 출력하는 프로그램을 구현하시오.

## 입력

첫 번째 줄에 중앙대학교 재학생의 수 N이 주어진다.

두 번째 줄에 N개의 투표 내역이 공백으로 구분되어 주어진다. 각각 찬성은 `1`, 반대는 `-1`, 기권은 `0`으로 주어진다.

## 출력

투표가 통과되었으면 `APPROVED`, 통과되지 않았으면 `REJECTED`, 무효 처리되었으면 `INVALID`를 출력한다.

## 제한

- 1 ≤ N ≤ 100

## 서브태스크

| 번호 | 배점 | 제한 |
| --- | --- | --- |
| 1 | 40 | 기권한 사람이 존재하지 않는다. |
| 2 | 60 | 추가적인 제약 조건이 없다. |

## 예제 입력 1

```
5
1 -1 1 1 -1

```

## 예제 출력 1

```
APPROVED

```

## 예제 입력 2

```
3
-1 1 -1

```

## 예제 출력 2

```
REJECTED

```

## 예제 입력 3

```
4
1 0 -1 1

```

## 예제 출력 3

```
APPROVED

```

해당 예제는 서브태스크 1에는 주어지지 않음을 유의하시오.

## 예제 입력 4

```
4
1 0 0 1

```

## 예제 출력 4

```
INVALID

```

해당 예제는 서브태스크 1에는 주어지지 않음을 유의하시오.
