## 문제

줄의 개수를 세는 프로그램을 작성하시오.

## 입력

한 줄에 최대 100글자씩 주어진다. 빈 줄이 주어질 수도 있다.

## 출력

입력으로 몇 줄이 주어졌는지를 출력한다.

## 예제 입력 1

```
one
and two
three

```

## 예제 출력 1

```
3
```
