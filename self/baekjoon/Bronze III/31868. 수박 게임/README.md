## 문제

1단계 과일은 체리, N단계 과일은 수박이다.

i단계 과일 2개를 소모하여 i + 1단계 과일 1개를 만들 수 있다. (1 ≤ i ≤ N − 1)

K개의 체리로 최대 몇 개의 수박을 만들 수 있는지 구해보자!

## 입력

첫 번째 줄에 정수 N과 K가 공백을 사이에 두고 주어진다. (2 ≤ N ≤ 30; 1 ≤ K ≤ 10^9)

## 출력

첫 번째 줄에 만들 수 있는 수박의 최대 개수를 출력한다.

## 예제 입력 1

```
3 10

```

## 예제 출력 1

```
2
```
