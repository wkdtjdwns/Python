## 문제

∑ 1 ≤ a < b ≤ n xa xb

## 입력

n과 xi가 주어짇나. n은 10만 이하ㅇ고, xi는 젗ㄹ댓값이 100이하인 정수디이다.

## 출력

위에서 구하란 걸 구하면 된ㄷ.

## 예제 입력 1

```
3
1 -2 3

```

## 예제 출력 1

```
-5
```
