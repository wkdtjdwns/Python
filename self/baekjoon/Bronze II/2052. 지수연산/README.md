## 문제

양의 정수 𝑁이 주어질 때, 1 / 2^𝑁을 계산하는 프로그램을 작성하시오.

## 입력

첫째 줄에 정수 𝑁이 주어진다.

## 출력

첫째 줄에 1 / 2^𝑁의 정확한 값을 소수 형태로 출력한다. 단, 뒤에 불필요한 0을 덧붙여서는 안 된다.

## 제한

- 1 ≤ 𝑁 ≤ 250

## 예제 입력 1

```
5

```

## 예제 출력 1

```
0.03125
```
