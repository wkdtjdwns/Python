## 문제

0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

## 입력

첫째 줄에 정수 N(0 ≤ N ≤ 100,000)이 주어진다.

## 출력

첫째 줄에 N!을 출력한다.

## 예제 입력 1

```
10

```

## 예제 출력 1

```
3628800

```

## 예제 입력 2

```
0

```

## 예제 출력 2

```
1

```

## 노트

PyPy도 Python이다. Python 3 유저는 PyPy3으로 제출하면 된다.
