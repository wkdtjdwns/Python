## **문제 설명**

```
본 문제는 python 의 빠른 기초 학습을 위해 설계된 문제로서 python 코드 제출을 기준으로 설명되어 있습니다. 
------

줄을 바꿔 문자(character) 2개를 입력받고, 순서를 바꿔 한 줄씩 출력해보자.
```

## 입력

```
2개의 문자가 줄을 바꿔 입력된다.
```

## 출력

```
순서를 바꿔 한 줄에 한 문자씩 출력한다.
```

## 입력 예시

```
a
b
```

## 출력 예시

```
b
a
```
